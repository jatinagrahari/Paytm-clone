## 🚀 Quick Start

### Prerequisites
- Modern web browser
- VS Code (recommended)

### Installation

#### Project Setup

```bash
# Clone the repository
git clone <your-repository-url>

# Navigate to project directory
cd Paytm-clone-main

# Open project
open index.html
```
